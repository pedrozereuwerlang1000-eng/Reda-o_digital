#  Simulador de Redação Digital

Este é um aplicativo desenvolvido em React para prática de redações no modelo oficial de 30 linhas.

##  Funcionalidades
- **30 Linhas Reais:** Alinhamento preciso sobre a pauta azul.
- **Caligrafia Realista:** Opções de letra Cursiva Clássica, Bastão Manual e Digitada.
- **Preto Puro:** Texto configurado em preto para máxima legibilidade.
- **Download em PDF:** Gera um arquivo A4 pronto para impressão sem cortes de acentos.

##  Como usar
1. Digite o título e o seu texto no editor lateral.
2. Escolha o estilo de letra que prefere (Cursiva ou Bastão).
3. Clique em **"Baixar PDF"** para salvar a folha pronta.

##  Tecnologias
- React.js
- Tailwind CSS
- Lucide React (Ícones)
- Html2Pdf.js (Gerador de documentos)
